#1 Personalia page

In case it's lost and found

The first page is a place for you to enter your personalia so it can be returned to you if lost and found. It also has a section for writing down some over-all notes or instructions.

The book contains 100 projects and 30 note pages. This ToC is a table with 130 lines, one entry per project and note page. It's a good idea to describe the contents of each page here, so you can easily get back to it at a later time.
